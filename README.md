# Golden Aura Studio
**By Stephie667**

## Project Description
Golden Aura Studio is a photography website that not only captures images but also captures the feeling of a moment. It displays both the unposed and raw moments, helping you turn your fleeting seconds into lifelong heirlooms.

## Features
* **Custom Branding:** High-impact "Golden Aura" aesthetic.
* **Responsive Layout:** Works on mobile and desktop.
* **Portfolio Gallery:** Professional display of photography moments.
